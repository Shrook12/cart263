## Description

This is a small interactive drawing app. It includes a pencil, eraser, ink, and a brush that can change size and color depending on what the user wants. You can also drag and drop some textures or images into the canvas. There are some challenges that you can do. Every couple of minutes, a transparent black screen will appear  to do a small physical exercise. This is to make you move more for your health since artists can sit for hours without moving. At the end, you can save your work!
